# MiniChat
A simple chat plugin for chat formatting

Features:

- MiniMessage support
- Channels:
  - Global
  - Global (Cross-server)
  - Local + Global
  - Local + Global (Cross-server)
- PlaceholderAPI support
- A (very) simple setup

Requirements:

- TwilightLib
- PlaceholderAPI
- SuperVanish/PremiumVanish (optionally)
